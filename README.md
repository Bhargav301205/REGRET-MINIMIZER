# Regret Minimizer AI 🧠

A smart decision-making system that predicts whether you will regret a choice using AI-inspired logic.

---

## 🚀 How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run backend:
python app.py

3. Open frontend:
Open frontend.html in browser

---

## 💡 Features

- Hybrid AI + rule-based decision system
- Regret scoring model
- Real-time UI interaction
- Safety guard layer for harmful inputs

---

## ⚠️ Important

Backend must be running (`python app.py`) before using the frontend.

---

## 🏆 Project Goal

To help users make better decisions by minimizing future regret using intelligent analysis.
